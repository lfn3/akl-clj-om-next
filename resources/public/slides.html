What's figwheel?

A cljs library for hot code loading

The magic is it keeps app state between reloads

Great with any app that keeps it's state in one place

What's devcards?

Put all your app's states on one page

Let's you get started and experiment way quicker

Great for testers, stakeholders etc.

Both of these can work outside the cljs ecosystem

... but they're less useful, and more finnickity.
